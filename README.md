# showbq
Golang bigquery examples

Before running command, 

1. Get the JSON credantials file : 
  Google Developers Console -> 
  API Console -> 
  Credentials -> 
  Add Credentials -> 
  Add Service Account -> 
  Download JSON key

2. Save it as g.pem under besides each go file.
